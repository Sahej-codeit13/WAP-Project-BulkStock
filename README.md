# BulkStock - Product Dashboard

## Project Overview
BulkStock is a simple web application made to display and manage product data.  
It fetches product details from an API and shows them in a clean dashboard format.  
The main idea of this project is to practice JavaScript concepts like API handling, array functions, and DOM manipulation.

---

## Features

- Fetch data from a public API  
- Display products on the screen  
- Search products by name  
- Filter products based on category  
- Sort products by price  
- Dark mode option  
- Add products to favorites (stored in local storage)  
- Responsive design (works on mobile and desktop)  

---
