# BulkStock - Product Dashboard

## Project Overview

BulkStock is a simple web application made to display and manage product data.
It fetches product details from an API and shows them in a clean dashboard format.
The main idea of this project is to practice JavaScript concepts like API handling, array functions, and DOM manipulation in a simple and practical way.

---

## Features

* Fetch product data from API
* Search products by name
* Filter products by category
* Sort products by price
* Dark mode toggle
* Add to favorites (LocalStorage)
* Loading indicator
* Responsive design

---

## Technologies Used

* HTML
* CSS
* JavaScript
* DummyJSON API

---

## API Used

https://dummyjson.com/products

2. Open the project folder
3. Open `index.html` using Live Server
